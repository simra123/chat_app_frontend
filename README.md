# Built with
~ React js  <br>
~ Custom Sass  <br>
~ Context Api  <br>

~ React Hooks  <br>

src > useContext > all global State <br>
src > views > reuseble > all components blocks <br>
src > views > chat > main chat page <br>
src > views > form > name Form <br>

# To Run
